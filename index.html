Project: Meet App

The Meet App is a serverless, progressive web application (PWA) built with React using a test-driven development (TDD) approach. It integrates with the Google Calendar API to allow users to filter events by city and number, visualizing data via interactive charts.

Key Features

  • Serverless Architecture: Utilizes AWS Lambda for backend authorization.
  • Google Integration: OAuth2 authentication and Calendar API data fetching.
  • Data Visualization: Interactive Pie and Scatter charts using Recharts.
  • PWA & Offline: Service workers enable offline functionality and home screen installation.
  • Testing: 90%+ coverage using Jest, Enzyme, and Cucumber for BDD.

Technologies Used

React • Recharts • AWS Lambda • Jest • Cucumber • Vite

Project: Chat App (Mobile)

A native mobile chat interface built with React Native. Users can enter chat rooms, share images from their library or camera, and share their real-time location. It features robust offline storage to keep conversations accessible without an internet connection.

Key Features

  • Native Mobile: Built for iOS and Android using React Native and Expo.
  • Real-time Database: chat history stored in Google Firestore.
  • Media Sharing: Custom actions for sending images and geolocation data.
  • Accessibility: Optimized for screen readers and accessibility standards.

Technologies Used

React Native • Expo • Google Firestore • Firebase Storage
